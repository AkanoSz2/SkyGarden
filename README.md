# Environment Variables

Create a `.env` file in the project root and add the following variables:

```env
VITE_HYPIXEL_API_KEY=017abd43-b22f-466c-8b25-04a4c54ca5d0
DISCORD_BOT_TOKEN=your_discord_bot_token_here
```

## Variables

- `VITE_HYPIXEL_API_KEY` – Your Hypixel API key used by the frontend.
- `DISCORD_BOT_TOKEN` – Your Discord bot token used for Discord integration.
